# KJMcGuire.com

https://kjmcguire.com

## Dependencies

 * Hugo
   * Download latest `hugo_extended` for your platform
   * https://github.com/gohugoio/hugo/releases

## Build

```
git clone git@github.com:KJMcGuire/kjmcguire.com.git
cd kjmcguire.com

git submodule init
git submodule update

hugo serve
```
